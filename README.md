# DOMIX AI MVP

Нулевой MVP для интерьерных заявок и локального подбора мебели по фото комнаты.

## Локальный редактор

Откройте файл `index.html` в браузере. Редактор работает без сервера, интернета и платных API.

Что умеет:

- загрузить фото комнаты;
- добавить мебель из простого каталога;
- двигать мебель мышью;
- менять цвет, размер и поворот;
- сохранить результат в PNG.

## Что умеет

Telegram-бот:

- принимает фото комнаты;
- спрашивает задачу, стиль и уровень бюджета;
- выдает дизайн-задание;
- генерирует промпт для будущего AI-визуала;
- при наличии `ADMIN_CHAT_ID` отправляет заявку админу.

## Почему без вложений

В этой версии нет платного AI API, базы данных и хостинга. Бот работает локально на вашем компьютере и использует только бесплатный Telegram Bot API.

## Запуск

1. Создайте бота через Telegram `@BotFather`.
2. Скопируйте токен.
3. В PowerShell задайте токен:

```powershell
$env:TELEGRAM_BOT_TOKEN="ваш_токен"
python bot.py
```

Опционально можно задать `ADMIN_CHAT_ID`, чтобы получать заявки себе:

```powershell
$env:ADMIN_CHAT_ID="ваш_chat_id"
```

Чтобы в боте появилась кнопка редактора, задайте `SITE_URL`:

```powershell
$env:SITE_URL="http://127.0.0.1:8765/index.html"
```

Для реальных пользователей нужна публичная HTTPS-ссылка, например GitHub Pages или Netlify:

```powershell
$env:SITE_URL="https://yourname.github.io/design-room/"
```

## Как тестировать

1. Откройте своего бота в Telegram.
2. Нажмите `/start`.
3. Отправьте фото комнаты.
4. Выберите задачу, стиль и бюджет.
5. Получите готовое дизайн-задание и промпт.

## Следующий шаг

Когда MVP подтвердит интерес пользователей, можно подключить генерацию изображений через AI API или локальную модель.
